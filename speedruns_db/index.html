Welcome to Speedruns DB!

This site is dedicated to keeping track of speedruns for select N64 games. This includes not only listing the top runs for each game, but also outlining the routes and giving explanations for the glitches and tricks that make these runs possible.

Our goal is to make it easy for new runners to get started and to help anyone have fun enjoying these games!

Games:

  • Super Mario 64
  • Zelda: Ocarina of Time
  • Zelda: Majora's Mask
  • Donkey Kong 64
  • Diddy Kong Racing
  • Banjo Kazooie
  • Banjo Tooie
